# Render Specification — Iron / Structural Steel (.weldi)

This document will define how WMDB Iron weld maps are rendered visually.

## Status

This render specification has not yet been defined. This file is a placeholder.

## Notes

The iron rendering rules (prefix characters, cell types, merging behavior, color policy) will be defined here once the iron drawing standard is established. Structural steel drawings may differ significantly from the boiler grid-based layout.

## Legend

All rendered iron drawings **must include a legend** explaining the prefix characters used for weld types. The legend must appear in the margins or footer of the drawing. Specific prefix characters will be defined when this standard is established.
