# Render Specification — Pipeline (.weldp)

This document will define how WMDB Pipeline weld maps are rendered visually.

## Status

This render specification has not yet been defined. This file is a placeholder.

## Notes

The pipeline rendering rules (prefix characters, cell types, merging behavior, color policy) will be defined here once the pipeline drawing standard is established. Pipeline drawings may differ significantly from the boiler grid-based layout.

## Legend

All rendered pipeline drawings **must include a legend** explaining the prefix characters used for weld types. The legend must appear in the margins or footer of the drawing. Specific prefix characters will be defined when this standard is established.
